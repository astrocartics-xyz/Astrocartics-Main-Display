# Astrocartics Main Display

Astrocartics Main Display is an open-source space visualization and display system. This project provides interactive dashboards, data visualization, and display tools for cartographic applications.

## Features

- Interactive dashboard for exploring space-based data
- Visualization components for astronomical and cartographic datasets
- Modular architecture for easy extension
- Real-time and static data display support

## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/astrocartics-xyz/Astrocartics-Main-Display.git
   cd Astrocartics-Main-Display
   ```

2. **Serve the project locally using [http-server](https://www.npmjs.com/package/http-server):**

   If you have [Node.js](https://nodejs.org/) installed, you can use `npx` to quickly run a local server.

   ```bash
   npx http-server
   ```

   By default, this will serve files from the current directory at [http://localhost:8080](http://localhost:8080).

   > If your main entry file is `index.html`, it will be picked up automatically.

3. **Open your browser and navigate to [http://localhost:8080](http://localhost:8080) to view the display.**

## Usage

- Explore interactive dashboards and visualizations
- Customize visual display components as needed
